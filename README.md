# FakeEthereum
Centralized EVM RPC API

1. Install requirement modules.
2. Change coin chain_id in rpc.py (optional)
# ![alt text](https://i.imgur.com/H4ZQymC.png)
3. Run rpc.py
4. Run block_miner.py
5. Add your network to Metamask.
6. Enjoy :)
